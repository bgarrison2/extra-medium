#Helpful tools for extracting espresso
There are lots of tools espresso nerds use for brewing espresso at home that are used by almost no coffee shops. There are a couple good reasons for this:

* If you only pull a couple shots a day, you don't build up the muscle memory that professional barristas have. And you have plenty of time, so why not use a few tools to optimize the workflow.
* A lot of these tools are gimmicks and you don't need them.

I've tried a few fancier ways to prep coffee in my portafilter, but I think these tools are the basics you need as a home espresso drinker and nothing more.
