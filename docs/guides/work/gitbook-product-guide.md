#Gitbook implementation with multiple sites and authentication
I currently have this Gitbook site implemented in my current role. It takes advantage of multiple spaces in a single site, all topics are synced with our Github repo, and it leverages Gitbook's cool AI Assistant.

![Gitbook Product Guide](./assets/gitbook-product-guide-example.png "Gitbook product guide example")


##Details
* Visitor authentication set up so that users can access from our product
* AI Assistant enabled
* Synced with Github
* Product Guide and Release Information are separate spaces that are included in a site
